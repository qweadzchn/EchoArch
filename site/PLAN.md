# 百泉古建筑群网站计划

## 已完成

### 站点结构

- 已完成 `React + TypeScript + Vite` 前端搭建
- 已建立素材同步脚本
- 已接入总览图、点位和建筑详情内容

### 浏览体验

- 首页已改成总览图进入
- 点位悬停、预览、点击入景已打通
- 详情页已改成独立阅读页
- 首页与详情页切换已加入更顺滑的过渡

### 导游系统

- 已加入灯笼式入口
- 已加入耳语提示与非常驻面板
- 已支持路线化游览
- 已支持前情回声与低声问路
- 已支持导游入口自由拖动
- 已支持 `open_spot`
- 已支持 `go_home`
- 已支持 `select_route`
- 已增加运行时 JSON 配置 `public/guide-agent.config.json`
- 已提供本地可跑的后端脚手架 `server/guide-agent.ts`

## 当前阶段目标

当前阶段重点不是堆功能，而是把这三件事做稳：

1. 总览探索继续增强沉浸感
2. 导游更像伴游路引，而不是聊天框
3. 真实后端 API 跑通并可持续扩展

## 下一步建议

### 第一优先级：跑通后端

1. 配置 `.env.guide`
2. 启动本地导游后端
3. 打开 `public/guide-agent.config.json`
4. 把 `enabled` 改成 `true`
5. 验证前端能调用本地 `/api/guide`

### 第二优先级：内容深化

1. 给每个建筑补一段导游短讲
2. 给每条路线补导语与收束语
3. 给实景 / 古风 / 素描图补解释模板

### 第三优先级：玩法深化

1. 强化“报站 / 转场 / 下一站”三段式体验
2. 增加今昔对照与图像解读
3. 做更完整的开放世界路线切换

### 第四优先级：语音体验

1. 先做 TTS 可播放短讲
2. 再评估实时语音导游

## 备注

- 编辑配置时改 `public/guide-agent.config.json`，不要改 `dist/`
- 真实 OpenAI 密钥只放后端 `.env.guide`
- 这个项目更适合做“伴游式导览”，不是通用聊天应用
