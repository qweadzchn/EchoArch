# 百泉湖三维场景复原依据

更新日期：2026-06-25

## 已采用的资料

1. 项目总览图 `public/landing/overview.jpg` 与点位对照图 `public/landing/reference.jpg`
   - 用于确定卫源庙、安乐窝、清晖阁、船房、放鱼亭、碑廊、南大厅、跃进亭等景点的相对方位。
   - 该图是空间锚点来源，不作为精确 GIS 测绘成果。
2. 项目实景档案 `public/heritage/<spot-id>/`
   - 卫源庙照片用于校准灰石台基、风化红木、青灰瓦和蓝底牌匾。
   - 清晖阁照片用于校准湖中双层水阁、灰瓦、红木和石栏围合关系。
   - 跃进亭、放鱼亭、湖心亭、船房等照片用于区分亭、廊桥与临水建筑形制。
3. 项目原始文字资料 `../各建筑介绍/`
   - 用于景点导游讲解、建筑年代和空间关系描述。
   - 例如清晖阁位于湖中偏西南、船房位于清晖阁南侧、邵夫子祠位于湖西岸、放鱼亭位于北湖东岸。

## 资料复原与合理近似的边界

- 资料复原：景点相对方位、主要建筑类别、清晖阁双层形制、跃进亭三连亭桥、青灰瓦/红木/灰石主色、主要桥路连通关系。
- 合理近似：场景采用 `1 unit ≈ 1m` 的视觉比例估算；地形高度、湖岸曲线、树木密度、建筑精确开间尺寸和碰撞体并非测绘数据。
- 模块化近似：尚无精确 GLB 的建筑使用屋顶、柱、格窗、台基、台阶、牌匾等复用部件搭建；后续可按同一 `WorldSpotDefinition.model` 接口替换为测绘模型。
